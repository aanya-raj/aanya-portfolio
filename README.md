# portfolio

my corner of the internet: https://aanya-raj.github.io/aanya-portfolio/

tarot themed, because if recruiters are going to judge my fate anyway,
we might as well make it official.

built with react, vite, tailwind and framer motion. there's a small
flask api in `aanya-backend/` that i use on my own machine to edit
content. the live site runs fine without it.

## run it

```bash
cd aanya-portfolio
npm install
npm run dev
```

that's it. the site falls back to built-in content when the backend
isn't running, which is exactly how the deployed version works.
